# Screenshots

Folder ini berisi screenshot tampilan website AkademikaBot.

## Cara Menambahkan Screenshot:

1. Ambil screenshot dari 4 halaman berikut:
   - **login.png** - Halaman login
   - **chatbot-user.png** - Interface chatbot user
   - **admin-live-chat.png** - Admin live chat interface
   - **admin-dashboard.png** - Admin dashboard

2. Save file gambar ke folder ini dengan nama yang sesuai

3. Commit dan push ke GitHub:
```bash
git add screenshots/
git commit -m "Add website screenshots"
git push origin main
```

Gambar akan otomatis muncul di README.md utama.
